## Running Env:

* OS: Ubuntu 16.04
* Node version: 8.11

## Build and Run Steps:
* npm i
* npm run start

## To parse a csv file
1. Copy file to `data/` directory in this repo
2. Execute this app by running `npm run start` and provide a path to the file when prompted
3. Parsed file will be saved in `data` directory as `processed_CSV.csv`
